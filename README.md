# Splunk SOC Incident Lab

## Project Overview

This project simulates a Security Operations Center workflow using Splunk.

The lab was inspired by investigative practices and monitoring concepts I was exposed to during my industrial attachment at the Kenya Revenue Authority (KRA).

All scenarios in this project are independently created in a controlled training environment and do not
reflect the internal systems, tools or processes used by KRA.

The objective is to demonstrate alert monitoring, log analysis, and incident investigation using Splunk.

## Scope

This project includes multiple simulated incidents representing common SOC alert scenarios including:

- Authentication anomalies  
- Suspicious process execution  
- Potential phishing activity  
- Privilege-related activity  
- Suspicious network behavior
